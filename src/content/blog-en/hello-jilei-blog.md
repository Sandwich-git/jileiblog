# Hello, jilei.blog

This is my personal technical blog, built with **Astro** + **Cloudflare Pages** — zero-cost hosting with global CDN acceleration. Domain `jilei.blog`, covering tips and solutions for using **Claude** effectively.

## Why this blog?

After using Claude for over half a year, I've accumulated quite a few experiences and tricks. Many problems recur, and rather than solving them from scratch each time, it's better to document them:

1. **Knowledge consolidation** — Writing is the best way to learn
2. **Help others** — Someone out there might have the same questions
3. **Conversation logs** — Claude conversations themselves make great material

## Blog features

- **Bilingual**: Chinese by default, some posts available in English
- **Fully static**: Fast loading, zero JavaScript overhead
- **Near-zero cost**: Domain ~$12/year, hosting free
